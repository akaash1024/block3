CSS Box Model Demonstration

Box 1
box-sizing: content-box
This box uses the default `content-box` model. The width does **not** include padding or border.
Box 2
box-sizing: border-box
In this model, the total width **includes** content, padding, and border.
Box 3
box-sizing: content-box (default)
Notice how changing padding and border affects the final box size.
